üìö About Dijkstra's Algorithm

Dijkstra's algorithm is a graph search algorithm that finds the shortest path between nodes in a graph. It's widely used in network routing protocols and transportation planning.

How it Works:

  1. Initialize all distances to infinity, except the start node (distance = 0)
  2. Mark all nodes as unvisited
  3. Select the unvisited node with the smallest distance
  4. Update distances to all neighbors
  5. Mark current node as visited
  6. Repeat until destination is reached

Time Complexity:

  • With Priority Queue: O((V + E) log V)
  • With Array: O(V¬≤)
  • Where V = number of vertices, E = number of edges
